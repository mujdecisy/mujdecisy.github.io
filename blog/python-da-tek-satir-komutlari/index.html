çoklu değer atama

syntax

a,b,c = 1,2,3
a,b = b,a

hızlıca birden fazla değişkene değer atamak istiyor olabilirsin ve değişken tiplerinin bu atama işleminde hiçbir önemi yok.

appleCount, bananaColor, isOrangeInBag, = 4, "yellow", True 
print(f"{appleCount}, {bananaColor}, {isOrangeInBag}")

aynı örneği bir veya birden fazla objenin değerini atamak için de kullanabilirsin.

class fruit:
    color = None
    count = None
    isInBag = None
    
apple = fruit()
orange = fruit()
banana = fruit()

apple.count, banana.color, orange.isInBag, = 4, "yellow", True 
print(f"{apple.count}, {banana.color}, {orange.isInBag}")

iki değişkenin değerini geçici değişken kullanmadan değiştirmek isteyebilirsin.

soldAppleCount = 0
remainingAppleCount = 5
soldAppleCount, remainingAppleCount = remainingAppleCount, soldAppleCount 
print(f"{soldAppleCount}, {remainingAppleCount}")

if else

syntax

a = 1 if condition else 0

bir değer atarken atanan değeri bir koşula göre atamak istediğini düşünelim.

dividend = 15
divisor1 = 0
divisor2 = None
divisor3 = 2.7

result1 = dividend/divisor1 if divisor1 != 0 else "out of condition"
result2 = dividend/divisor2 if divisor2 != None else "out of condition"
result3 = dividend/divisor3 if isinstance(divisor3, int) else "out of condition"

print(f"{result1}, {result2}, {result3}")

hızlıca null-check, type-check yapabilirsin veya zero-division hatalarını bertaraf edebilirsin.

liste oluşturma

syntax

a = [process(i) for i in x]
b = [[process(i,j) for j in a] for i in a]
c = [j for i in b for j in i]

hemen istediğimiz uzunlukta tüm değerleri 2 olan bir liste oluşturalım. ardından bu listeyi manipule edelim. tek satırda bu listenin değerlerinin karesini veren yeni bir liste oluşturalım.

branchCountInCites = [2 for i in range(81)]
print(branchCountInCites)

branchCountGoalInCites = [e**2 for e in branchCountInCites]
print(branchCountGoalInCites)

peki ya çok boyutlu bir matris oluşturmak istiyorsak? ardından bu matrisimizi tek boyutlu bir listeye dönüştürelim.

set ve dictionary oluşturma

syntax

a = {process(i) for i in x}
b = {keyit(i) : valueit(i) for i in x}

verilen bir listeden küme oluşturmayı öğrenelim.

modulusByThree = [i%3 for i in range(10)]
uniqueModulus = {i for i in modulusByThree}
print(uniqueModulus)

verilen iki listeden bir dictionary oluşturalım.

clothes = ["skirt", "tshirt", "dress"]
prices = [12, 9, 17]

pricesByClothes = {clothes[i] : prices[i] for i in range(len(clothes))}
print(pricesByClothes)

filtreleme

syntax

a = [e for e in x if condition(e)]

verilen bir listeden bir koşula bağlı olarak yeni bir liste çıkartmamız gerektiğini düşünelim.

students = [i for i in range(10)]
studentsExaminingAtClassA = [e for e in students if e%2 == 0]
studentsExaminingAtClassB = [e for e in students if e%2 == 1]

print(students)
print(studentsExaminingAtClassA)
print(studentsExaminingAtClassB)
